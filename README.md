## Introduction

## Build & Run

You need to install these packages on your computer

- make
- gcc (or use clang, pass CC=clang to change)

```bash
make && make run
```

## Builtin functions/features

1. print, include\_native

2. +, -, \*, /, >, <, =

3. function, define, if-elseif-else
